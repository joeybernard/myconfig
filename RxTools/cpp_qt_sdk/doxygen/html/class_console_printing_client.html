Detailed Description

The ConsolePrintingClient class is an example SsnRx client displaying its output on the text console.

It displays messages on the textual console representing the communication with the receiver. This class defines a range of slots that are connected to the correponding signals defined in SsnRx, and that display the corresponding information to the user.
