6.1.2 Launching RxLogger

RxLogger can be launched in several ways: using the RxLauncher GUI, from the Start menu on a Window's PC, a shortcut to the RxLogger executable can be found under 'Septentrio RxTools'. You can also launch RxLogger via the 'Tools' menu of any of the GUI tools, for example from RxControl as shown in Figure 6-2. RxLogger can also be launched via the command console which allows to specify options as well. To know all possible options lauch RxLogger with the option "-h".
