3.1.2 Launching DataLink

Data Link can be launched in several ways: using the RxLauncher GUI, from the Start menu on a Window's PC, a shortcut to the Data Link executable can be found under 'Septentrio RxTools'. You can also launch Data Link via the 'Tools' menu of any of the GUI tools, for example from RxControl as shown in Figure 3-2.
