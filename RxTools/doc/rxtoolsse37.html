11.10 posconv

The RxTools installation contains the posconv tool which converts a given position between decimal degrees (-d), radians (-r) and cartesian coordinates (-c).

For example, the command below generates the output given. Note that there should be no spaces between the commas and the coordinates when using this tool.

posconv.exe -d 50.848,4.731,127.38 <CR>

 
 DATUM          :  WGS84  
 Geodetic(d.d)  :  Lat: 50.84799957  Lon: 4.73099995  Alt: 127.37999725  
 Geodetic(rad)  :  Lat: 0.887465010597  Lon: 0.082571525980  Alt: 127.379997253418  
 Cartesian (xyz):  X: 4021489.728496  Y: 332817.276668  Z: 4922984.447794
