9.1.2 Launching RxPlanner

RxPlanner can be launched in several ways: using the RxLauncher GUI, from the Start menu on a Window's PC, a shortcut to the RxPlanner executable can be found under 'Septentrio RxTools'. You can also launch RxPlanner via the 'Tools' menu of any of the GUI tools, for example from RxControl as shown in Figure 9-2.
