2.1.2 Launching RxControl

RxControl can be launched in several ways: using the RxLauncher GUI, from the Start menu on a Window's PC, a shortcut to the RxControl executable can be found under 'Septentrio RxTools'. You can also launch RxControl via the 'Tools' menu of any of the GUI tools, for example from SBF Analyzer as shown in Figure 2-2.
